　　在看完《学习JavaScript数据结构与算法》后，开始再刷 leetcode, 先按照书中的讲解的顺序，按照 tag 来刷几道简单的题目。刷了一个月了，还是很蒙蔽的状态。后来又买了一本《labuladuo的算法小册》，书中建议先刷二叉树。于是在刷了一些二叉树之后，发现理解递归尤为重要。二叉树大部分的题目无非都是在前序，中序，或者后序遍历的时候做些事情。在做递归的时候，总是忍不住人脑去递归，递着递着，人就傻了。所以总结下自己的对递归的理解。

不要陷入到递归

　　很多人开始再刷递归题目的时候，总是会想搞清楚函数之间到底发生了什么，这样很容易就陷入递归当中的递当中，已经归不回来了。直接懵逼了。
　　其实函数自身调用自己，就是压栈出栈的过程。计算机很容易处理这种事情。而人脑本身就不适合做这种事情。所以不要做不适合自己做的事情。吃力不讨好。

我该怎么做

　　把我们定义的函数看成一个数学函数。

f(x) = y;

　　我们要明确我们的函数到底会做什么？输入一个值，输出的就是一个我们想要的结果。然后通过控制参数，通过函数一步步的递进求得最后终止条件的值，然后再递归回来。

举几个例子

　　分别举例，一个简单题目，一个中等题目，一个困难题目。

简单

563.二叉树的坡度

计算每个节点的坡度，按照我们上一节讲的，明确我们的函数到底会做什么？

  1. 首先这道题目是求所有节点的坡度， 所以只要求得每个节点的坡度，最后再加起来就行
  2. 明确我们的递归函数是做什么 我们函数就是一个树的所有的值和
    const rec = (root) => {
  // 求一个树的和，左树 + 右树 + 节点本身
  return left + right + root.val;
}
  3. 函数还要做其他什么？先求每个节点的坡度
    const rec = (root) => {
  // 随便什么遍历，后续最好
  let left = rec(root.left)
  let right = rec(root.right)
  let 每个节点坡度 = Math.abs(left - right);
  // 求一个树的和，左树 + 右树 + 节点本身
  return left + right + root.val;
}
  4. 然后把每个节点加起来
    let sum = 0
const rec = (root) => {
  // 随便什么遍历，后续最好
  let left = rec(root.left)
  let right = rec(root.right)
  sum = sum + Math.abs(left - right);
  // 求一个树的和，左树 + 右树 + 节点本身
  return left + right + root.val;
}

    中等

    98. 验证二叉搜索树
  1. 首先这道题目是验证二叉搜素树，和题目描述的二叉搜素树的特征一样
  • 节点的左子树只包含小于当前节点的数。
  • 节点的右子树只包含大于当前节点的数。
  • 所有左子树和右子树自身必须也是二叉搜索树
  1. 明确我们的递归函数是做什么 我们函数就是判定一个树是不是二叉树，是就返还true,不是就返还false
    var isValidBST = function(root) {
  return isValidBST(root.left) && isValidBST(root.right);
};
  2. 函数还要做其他什么？ 上面的函数，只是求了 左子树 和 右 子树 是不是一个搜素二叉树。那么本身应该怎么判断？我们在前序遍历中，可以拿到当前节点的值。当前这个节点大于右子树的值，小于左子树的值，那么就符合条件。
    var isValidBST = function(root, min, max) {
  if(root.val >= min && root.val <= max) return {
    isValidBST(root.left, min, root.val) && isValidBST(root.right, root.val, max);
  }
  return false;
  3. 添加终止条件，当节点为空时，都满足，返回true;
    var isValidBST = function(root, min, max) {
  if(!root) {
    return true;
  }
  if(root.val >= min && root.val <= max) return {
    isValidBST(root.left, min, root.val) && isValidBST(root.right, root.val, max);
  }
  return false;

    复杂

    124.二叉树中的最大路径和
  1. 什么是最大路径和？左子树最大路径和加上右子树最大路径和加上节点本身
  2. 明确我们的递归函数做什么？如果只有一颗树，就两个节点，那么最大的路径就是节点本身加上左子节点和右子节点当中最大的那个
    const rec = (root) => {
  return root.val + Math.max(left,right)
}
  3. 函数还要做什么？对比所有的左子树和右子树，选最大的那个跟当前节点相加。
    var maxPathSum = function(root) {
    // 后续遍历
  let ans = Number.MIN_SAFE_INTEGER;
  const rec = (root) => {
    let left = Math.max(0, rec(root.left));
    let right = Math.max(0, rec(root.right));
    ans = Math.max(ans, left + right + root.val);
    return Math.max(left, right) + root.val;
  }
  rec(root);
  return ans;
};
  4. 添加终止条件
    var maxPathSum = function(root) {
    // 后续遍历
  let ans = Number.MIN_SAFE_INTEGER;
  const rec = (root) => {
    if (root == null) {
      return 0;
    }
    let left = Math.max(0, rec(root.left));
    let right = Math.max(0, rec(root.right));
    ans = Math.max(ans, left + right + root.val);
    return Math.max(left, right) + root.val;
  }
  rec(root);
  return ans;
};
